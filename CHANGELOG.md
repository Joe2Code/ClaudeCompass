# Changelog

## 1.0.0

Initial release.

- Menu bar usage indicator with color-coded pacing percentage
- Dashboard popover with daily stats (messages, sessions, tool calls, tokens)
- Weekly pacing with configurable reset day and hour
- Model breakdown (per-model message and token counts)
- Hourly distribution chart
- Weekly activity trend
- Optional web usage via Anthropic API (remaining messages, reset time)
- Configurable usage notifications with system permission handling
- Launch at login support
- Display modes: Weekly, Daily, Dual, Icon Only
- Credentials stored securely in macOS Keychain
- Built with Swift 6, strict concurrency, SwiftUI, and @Observable
